# PythonFromDAToDS
本项目为图书《Python编程：从数据分析到数据科学》的配套开源资源，包括：
  1.教学大纲，见文件夹“syllabus”
  2.PPT，见文件夹“PPTs”
  3.源代码，见文件夹“Codeds”
  4.数据文件，见文件夹“DataSets”
  5.软件工具，见文件夹"Tools"
  6.勘误表，见文件夹“Errata”
  7.练习题及其答案，见文件夹“Excercise”
  8.样章，见文件夹”SampleChapters“
  9.其他，见文件夹”Others“
